## Hapi + Sequelize 


RESTful actions for fetching, creating, updating and deleting 
```
GET     /api/authors
GET     /api/authors/:id
POST    /api/authors
PUT     /api/authors/:id
DELETE  /api/authors/:id

GET     /api/websites
GET     /api/websites/:id
POST    /api/websites
PUT     /api/websites/:id
DELETE  /api/websites/:id

GET     /api/articles
GET     /api/articles/:id
POST    /api/articles
PUT     /api/articles/:id
DELETE  /api/articles/:id

```
